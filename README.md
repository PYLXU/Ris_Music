# 瑞思音乐-Ris_Flood
## 简洁的消息发送器
由PYLXU(腾瑞思智)开发，在Github使用LGPLv3协议开源

[请点击这里下载最新版本的构建](https://github.com/PYLXU/Ris_Music/releases)

## 基本说明
**1.界面美观**

简洁而又美观大方的界面设计，让您的音乐之旅焕然一新

**2.稳定工作**

精心优化，减少您软件崩溃的次数和可能

**3.在线资源**

支持从网易云音乐获取在线资源，畅听网络

**特别提醒：本程序不提供VIP歌曲免费播放**
